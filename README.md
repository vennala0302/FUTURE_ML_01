# Sales & Demand Forecasting for Businesses Using Machine Learning

## Project Overview

This project predicts future sales demand using Machine Learning techniques. Historical sales data was analyzed and used to train a Linear Regression model to forecast future sales trends and support business decision-making.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Features

* Historical sales analysis
* Sales trend visualization
* Machine Learning forecasting
* Future sales prediction for the next 6 months
* Model evaluation using MAE, RMSE, and R² Score

## Generated Outputs

* Sales Trend Graph
* Actual vs Predicted Sales Graph
* Future Sales Forecast Graph

## Model Performance

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

## Business Insights

The forecasting model indicates a steady increase in future sales demand. Such predictions can help businesses improve inventory planning, resource allocation, and strategic decision-making.
